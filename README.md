# يوسوفت محاسب الموردين والمخزون والمبيعات

نظام محاسبي متكامل مصمم لإدارة عمليات المشتريات، المبيعات، المخزون الفعلي، وحسابات الموردين بدقة وكفاءة. يقوم النظام بتطبيق منطق مطابقة متقدم لتسوية المرتجعات وحساب التكاليف وصافي الأرباح، مع إنتاج تقارير مفصلة تدعم اتخاذ القرار.

---

## المميزات الرئيسية

-   **يعمل بدون اتصال بالإنترنت (Offline-First):** تصميم النظام ليعمل بكفاءة تامة دون الحاجة لاتصال بالإنترنت.
-   **معالجة البيانات غير التراكمية:** آلية استيراد ملفات جديدة تقوم بحذف السجلات القديمة لضمان عدم التضارب.
-   **أداء عالي:** بنية بيانات قادرة على التعامل مع حجم بيانات كبير يصل إلى 200,000 سجل.
-   **دعم كامل للغة العربية:** واجهة مستخدم متوافقة مع الاتجاه من اليمين إلى اليسار (RTL).
-   **واجهة مستخدم حديثة:** تصميم مرن باستخدام علامات تبويب للتنقل بين التقارير.
-   **تقارير قابلة للطباعة:** تخطيطات احترافية مناسبة للطباعة لكل تقرير.
-   **بحث وتصفية متقدم:** وظائف بحث سريعة وفعالة داخل الجداول.

---

## التقنيات المستخدمة

-   **[Electron](https://www.electronjs.org/):** لإنشاء تطبيق سطح مكتب متعدد المنصات.
-   **[React](https://reactjs.org/):** لبناء واجهة المستخدم التفاعلية.
-   **[Vite](https://vitejs.dev/):** كأداة بناء سريعة وحديثة.
-   **[Ant Design](https://ant.design/):** مكتبة مكونات واجهة مستخدم احترافية تدعم العربية.
-   **[Node.js](https://nodejs.org/):** بيئة تشغيل JavaScript.

---

## التثبيت والتشغيل

### المتطلبات الأساسية

-   تأكد من تثبيت **Node.js** (النسخة LTS) على جهازك. يمكنك تحميله من [هنا](https://nodejs.org/).

### خطوات التشغيل

1.  **تحميل المشروع:** قم بتحميل أو استنساخ مجلد المشروع `يوسوفت محاسب الموردين والمخزون والمبيعات` إلى جهازك.
2.  **فتح المشروع:** افتح المجلد باستخدام محرر الأكواد **Visual Studio Code**.
3.  **فتح Terminal:** من القائمة العلوية، اضغط `Terminal` > `New Terminal`.
4.  **تثبيت الحزم:** قم بتنفيذ الأمر التالي لتثبيت جميع المكتبات اللازمة:
    ```bash
    npm install
    ```
5.  **بدء التطبيق:** لبدء تشغيل التطبيق في وضع التطوير، قم بتنفيذ الأمر التالي:
    ```bash
    npm run electron-dev
    ```
    سيتم فتح نافذة تطبيق سطح المكتب تلقائيًا بعد بضع ثوانٍ.

## المنطق المحاسبي

يتبع النظام منطقًا محاسبيًا مفصلاً لمعالجة البيانات، يشمل:

-   **تجهيز صافي المشتريات والمبيعات:** باستخدام 8 مفاتيح مطابقة مرنة لاستنزال المرتجعات.
-   **تجهيز الجرد الفعلي:** معالجة الكميات السالبة والمنتهية الصلاحية.
-   **حساب المخزون النهائي:** مطابقة الجرد الفعلي مع صافي المشتريات.
-   **حساب تكلفة المبيعات:** ربط المبيعات بتكلفتها الفعلية.
-   **إعداد تقارير الموردين:** حساب الاستحقاقات وتحديد المخزون المعد للإرجاع.
