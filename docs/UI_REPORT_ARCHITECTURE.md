يصف هذا المستند المعايير التقنية والهيكلية المتبعة في بناء صفحات التقارير ضمن نظام YosSoft Accountant.
الهدف العام

تصميم واجهات مستخدم (UI) وتجربة مستخدم (UX) موحدة، عصرية، واضحة، وسهلة الاستخدام، مع الحفاظ التام على الدقة المحاسبية، بنية المشروع الأساسية، وأداء النظام.

ممنوع:

    تغيير أي معادلة محاسبية أو منطق حسابي.

    إضافة/حذف حقول بيانات دون موافقة فريق المحاسبة.

    التأثير على دقة الأرقام أو الإجماليات.

التسلسل الهرمي للمكونات

يتم ترتيب المكونات برمجياً وبصرياً وفق التسلسل التالي لضمان أفضل أداء وسهولة الصيانة:

    طبقة الترويسة الدلالية (Semantic Header Layer)

        المكونات: UnifiedPageLayout مع Typography.Title وTypography.Text.

        الصف الأول: عنوان التقرير + عدد السجلات (حجم 24px، وزن 700، لون #1976d2) مع أزرار الإجراءات (Print/Export/Clear Filters/View Settings).

        الصف الثاني: نبذة مختصرة (حجم 14px، لون #555555) من i18n/ar/report_descriptions.json.

    محرك الفلترة الذكي (Intelligent Filtering Engine)

        المكون: UnifiedFilterBar.

        يشمل: بحث شامل، فلاتر سياقية (رمز المادة، اسم المادة، المورد)، زر "مسح الفلاتر".

        تطبيق فوري مع تحديث فوري.

    جدول البيانات مع شريط التمرير العلوي (Data Table with Top Scrollbar)

        المكون: UnifiedTable - Grid Core مع SynchronizedHorizontalScrollbar.

        رأس ثابت (sticky، drag/drop، resize، sorting).

        جسم تفاعلي (highlighting، تنسيق أرقام: #,##0.00 د.ك، أحمر للسالب).

        شريط تمرير أفقي علوي متزامن.

واجهة التحكم في الترقيم (Pagination Controls)

    المكون: UnifiedTable - Pagination.

    منتقي حجم الصفحة (50-500)، أزرار التنقل، مربع "Go to Page" مع زر.

نظام التمرير المتزامن السفلي (Bottom Synchronized Scroll)

    المكون: SynchronizedHorizontalScrollbar.

    شريط تمرير أفقي سفلي مزامن.

    طبقة تفسير التقارير (Interpretation Layer)

        المكون: UnifiedPageLayout - Interpretation Container.

        قسم قابل للطي من docs/report_notes_ar.md.

    نهاية الصفحة

        حقوق الملكية: © YosSoft 2025 | YosSoft Accountant Platform.
